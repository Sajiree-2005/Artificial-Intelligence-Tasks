# Artificial Intelligence Projects Portfolio

A collection of **Artificial Intelligence projects** demonstrating practical applications in **Machine Learning**, **Natural Language Processing (NLP)**, and **Computer Vision** using Python.

---

## 🚀 Projects

### 📧 Spam Email Classifier
**Objective:** Classify emails as spam or not spam.

**Implementation Details:**
- TF-IDF Vectorizer for text features
- Multinomial Naive Bayes model
- Dataset split into training and testing sets
- Pipeline for streamlined workflow

**Key Features:**
- Predicts spam vs. ham emails
- Evaluates model accuracy
- Handles custom user input messages

---

### 😀 Face Detection System
**Objective:** Detect human faces in images.

**Implementation Details:**
- OpenCV with Haar Cascade Classifier
- Images converted to grayscale for better detection

**Key Features:**
- Detects multiple faces in an image
- Draws bounding boxes around detected faces
- Saves processed output images

---

### 💬 Sentiment Analysis on Text Data
**Objective:** Classify text sentiment as positive, neutral, or negative.

**Implementation Details:**
- Text preprocessing: lowercasing, regex cleaning, stopword removal (NLTK)
- TF-IDF for feature extraction
- Naive Bayes for model training

**Key Features:**
- Multi-class sentiment classification
- Comparison with TextBlob sentiment analysis
- Custom prediction function for user inputs

---

### 🤖 Rule-Based Chatbot
**Objective:** Build an interactive chatbot using rule-based logic.

**Implementation Details:**
- Developed in Python
- GUI built with Tkinter
- Conditional logic (if-else) for responses

**Key Features:**
- Handles greetings and FAQs
- Interactive chat interface
- Exit functionality and auto-scroll chat window
